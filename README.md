# 💖 Valentine-Themed Proposal Web App

A beautiful, interactive, and responsive single-page web application designed to playfully ask someone special to be your girlfriend. It features a step-by-step interactive questionnaire, custom user inputs, and a playful "runaway" 'No' button that guarantees a 'Yes'!

---

## ✨ Features

*   **Interactive Flow:** Walks the user through fun multiple-choice and text-input questions before delivering the final big question.
*   **The "Runaway" 'No' Button:** On the final screen, the 'No' button actively avoids both desktop cursor hover (`mouseover`) and mobile touch (`touchstart`), making it playfully impossible to click.
*   **Aesthetic UI:** Designed with a romantic Valentine's theme featuring a soft pink gradient background, floating CSS hearts, elegant typography (*Dancing Script* & *Quicksand*), and modern glass-morphism cards.
*   **Celebration Screen:** Clicking 'Yes' triggers an animated confetti burst along with a playful prompt asking her to send a text confirmation since no backend database is required.
*   **Fully Responsive:** Built with Tailwind CSS, ensuring a seamless experience across mobile, tablet, and desktop screens.

---

## 🛠️ Built With

*   **HTML5** – Page structure and content semantics.
*   **Tailwind CSS** – Contemporary utility-first styling.
*   **Vanilla JavaScript** – Multi-step screen navigation, runaway button logic, and state management.
*   **Canvas-Confetti** – Lightweight, external JavaScript library for the celebration effect.

## 🚀 Getting Started / Deployment

Since this is a lightweight, frontend-only static webpage, setting it up is incredibly simple:

### Local Setup
1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.

### Free Hosting Options
To share this link with her, you can host it online for free using platforms like:
*   **GitHub Pages:** Push the code to a GitHub repo, go to **Settings** -> **Pages**, and enable it.
*   **Netlify / Vercel:** Drag and drop the project folder directly onto their deployment dashboards.
*   **InfinityFree:** Upload via FTP to your standard web directory.

## 📜 Customization

If you want to tailor the text or change the sequence:

Questions: Edit the text blocks inside the id="step-1", id="step-2", and id="step-3" divs inside the index.html file.

Styling: Adjust colors and animations using the Tailwind class syntax embedded within the file.
